UltraBasic Series (1XXX)
Nokia 1112
UltraBasic Series
The Nokia 1112 was a lightweight, ultra-basic phone with a monochrome display, 700 mAh battery, and iconic Snake Xenzia. Perks included speaking clock, picture messaging, polyphonic ringtones, and swappable covers—perfect for first-time users.
Nokia 1661
UltraBasic Series
The Nokia 1661 was a durable, ultra-basic phone with a 1.8" color screen, FM radio, and flashlight. Perks included a speaking clock, 500-contact phonebook, polyphonic ringtones, and a long-lasting 860 mAh battery—perfect for everyday essentials.
Basic Series (2XXX)
Nokia 2100
Basic Series
The Nokia 2100 was a compact GSM phone with a monochrome display, 720 mAh battery, and iconic games like Snake II. Perks included animated screensavers, picture messaging, Xpress-on covers, and a photo slot on the back.
Youth Expression Series (3XXX)
Nokia 3100
Youth Expression Series
The Nokia 3100 featured a 1.5" color screen, Java games like Snake EX2, and GPRS internet. Perks included glow-in-the-dark covers, animated wallpapers, polyphonic ringtones, and Pop-Port support for add-ons like a camera or light shell.
Nokia 3105
Youth Expression Series
The Nokia 3105 was a CDMA version of the 3100 with a built-in flashlight, voice commands, and Java games. Perks included glow-in-the-dark covers, FM radio, scheduler, and support for camera and voice recorder via Pop-Port.
Nokia 3120
Youth Expression Series
The Nokia 3120 was a refined version of the 3100 with a sleeker design, 1.6" color screen, and Java support. Perks included SmileyWorld themes, animated screensavers, and optional Pop-Port camera—businesslike with a playful twist.
Nokia 3220
Youth Expression Series
The Nokia 3220 had a VGA camera, 65K color screen, and EDGE internet. Perks included LED light effects, wave messaging, custom cut-out covers, Java games, and optional NFC shell—making it a fun, flashy favorite.
Nokia 3310
Youth Expression Series
The Nokia 3310 was a legendary GSM phone with a monochrome screen, 900 mAh battery, and iconic Snake II. Perks included custom ringtones, picture messaging, Xpress-on covers, and unmatched durability—earning its cult status.

Nokia 6700 Classic

Nokia 7260
Youth Expression Series
The Nokia 7260 was a fashion-forward phone with a 1.6" color screen, VGA camera, and EDGE data. Perks included FM radio, infrared, Java games, and a bold silver swirl design—style and substance in one sleek bar.

The Nokia 7250i featured a CIF camera, 1.5" color screen, and Java apps. Perks included FM radio, infrared, xHTML browser, Xpress-on covers, and photo caller ID—blending fashion and function in a compact frame.
